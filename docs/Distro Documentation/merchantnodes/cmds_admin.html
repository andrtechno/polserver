Merchant and Guard Node - Admin Commands

For the merchantnodes package in the Distro, by Yukiko.

Administrator Commands

This is a list of the administrator (Admin) level commands and their description:
grabnodes
 This command is very useful in the event you need to perform an item wipe because it is used to save all nodes, both merchant and guard nodes, and merchant and guard data on those nodes located in the realm the user is in. To save the nodes in each realm the user must teleport to the realms and issue this command whilst in that realm. The node data is saved to a config file in /pkg/systems/spawnSystem/config/
Usage is .grabnodes [file name] where file name is optional. If the user does not enter a file name the command will default to "packedNodes-" with the realm name appended. If a file name is specified then -<realm name> is appended to the name entered by the user.
seednodes
 This is the opposite of grabnodes used to restore grabbed nodes from a config file created by the .grabnodes command. The nodes will be restored to the realm the user is in. To restore the nodes in each realm the user must teleport to the realms and issue this command whilst in that realm.
