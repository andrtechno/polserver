Sysbook package Documentation

For the sysbook package in the Distro, by Yukiko.

Game Master Commands

addspawnshelf
 Allows a user to add a bookshelf to the spawn shelf data file. If the user appends a <space> followed by an integer, the integer will be used to set the maximum books that will be spawned in the bookshelves targetted. If no integer is appended to the command a gump will be presented with the default value from the sysbook settings file. The user can change it if so desired.
createshardbook
 This programme displays a list of all books stored in the Shard Library (shard_library data file). There are check boxes beside each title to allow the staff member to create the books selected.
removespawnshelf
 This command allows the user to remove a bookshelf from the spawnedbookshelves data file.
