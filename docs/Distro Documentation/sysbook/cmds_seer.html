Sysbook Documentation

For the sysbook package in the Distro, by Yukiko.

Seer Commands

bookro
 Use this command to set a book as read only.
setbook
 This command allows you to change a book's graphic and its read/write status.
setpages
 This command allows the user to modify the maximum number of pages in a book. Please note: Due to the unique way books work in UO the maximum number of pages will always be an odd number or else the last page, on the right hand side will be blank. If the user enters 48 as the maximum pages this command will add 1 to that number to make it 49.
